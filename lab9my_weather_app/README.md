# 🌦 SkyCast - Weather App (Lab 9)

แอปพลิเคชันรายงานสภาพอากาศที่พัฒนาด้วย Flutter โดยดึงข้อมูลผ่าน Open-Meteo API

## 📌 สถานที่ที่รองรับในแอป
1. **Uttaradit (อุตรดิตถ์)** - พิกัด 17.62, 100.09
2. **Nan (น่าน)** - พิกัด 18.78, 100.77
3. **Chiang Mai (เชียงใหม่)** - พิกัด 18.79, 98.98
4. **Doi Lo (ดอยหล่อ)** - พิกัด 18.47, 98.78

## 🛠 ฟีเจอร์หลัก
- แสดงอุณหภูมิและความเร็วลมแบบ Real-time
- มี Animation หมุนไอคอนตลอดเวลาเพื่อความสวยงาม
- ใช้หน้าจอแบบ Gradient ไล่สีโทนม่วง-ฟ้า (Deep Purple & Blue Accent)
